# ShoppingGPT

ShoppingGPT is a GenAI-powered ecommerce recommendation assistant that helps users discover products using natural language.

Instead of manually filtering products, users can simply describe what they need.

## Features

- Semantic product search
- AI-generated recommendations
- Product comparison
- Vector similarity retrieval
- Product reranking
- Natural language shopping assistant

## Example Queries

- Best gaming laptop under 1 lakh
- Budget smartwatch with AMOLED display
- Minimal sneakers for office and casual wear
- Headphones for coding and meetings
- Gift ideas for a software engineer

## Tech Stack

- Streamlit
- FAISS
- Sentence Transformers
- OpenAI
- DuckDB
- Python

## Architecture

1. Product ingestion
2. Embedding generation
3. Vector indexing
4. Semantic retrieval
5. Product reranking
6. LLM recommendation generation
